BFS (Breadth First Search)	Facts: BFS can be started from any vertex you like also while exploring vertices you can visit vertices in any order you like.<br><br>Rule is:<br>1. When you are selecting a vetex for exploration then you should visit all its adjacent verices then only you should go to next vertex for exploration<br>2. You should select the next vertex for exploration from a queue data structure only.<br><br>BFS spanning tree<br>Level order of binary tree<br>Time complexity: O(n) where n is no of vertices
DFS (Depth First Search)	Facts: DFS can be started from any vertex you like.<br><br>Rule is:<br>1. While exploration once you have reached a new vertex, start exploring it and suspend the previous one , similarly you dig deep till no new node to explore. <br>2. Suspended node to be stored in stack data structure which will be used while coming back during traversal.<br><br>DFS spanning tree<br>Preorder<br>Time complexity: O(n) no of vertices
